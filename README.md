# CIS 5 – Assignment 3: Student Score Analysis

## Scenario
This program simulates a teacher entering student scores to analyze class performance.

## Program Features
- Uses an array of 10 student scores (0–99)
- Calculates the sum and average
- Determines pass/fail status (70 or higher)
- Finds the highest and lowest score without sorting the array

## Logic Explanation
The program loops through the array once to:
- Collect scores
- Track the sum
- Compare values to determine highest and lowest scores
Conditionals are used to determine pass or fail status.
